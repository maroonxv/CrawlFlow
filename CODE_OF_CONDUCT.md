# 行为准则

## 我们的标准

我们希望 CrawlFlow 对所有人来说都是一个尊重他人、友好开放的项目。

鼓励的行为包括：

- 在讨论和评审中保持尊重。
- 提供具体且有建设性的反馈。
- 在保持清晰和坦诚的同时，默认他人出于善意。
- 主动帮助其他人理解代码库与协作流程。

不可接受的行为包括：

- 骚扰、侮辱或歧视性言论。
- 人身攻击或故意威胁恐吓。
- 未经允许公开他人的私人信息。
- 反复打断、带偏或破坏技术讨论。

## 适用范围

本行为准则适用于仓库讨论区、Issue、Pull Request 以及其他所有贡献者以 CrawlFlow 名义参与的项目空间。

## 执行方式

当行为违反本准则时，项目维护者可以删除相关评论、拒绝相关贡献，或采取其他适当的管理措施。

## 举报方式

如果你遭遇或目睹了不可接受的行为，请通过私下渠道联系维护者，而不是直接在公开讨论串中处理。
